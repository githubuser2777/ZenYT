# ZenYT

ZenYT is a lightweight, stand-alone desktop application that provides a beautiful, premium User Interface for `yt-dlp`. 

## Features
- Fully harnesses the power of `yt-dlp`.
- Download video/audio, playlists, channels, subtitles, and metadata.
- Lightweight: Built with Tauri (Rust) and React.
- Premium UI: Smooth, modern, and carefully crafted using Vanilla CSS.
