# Application Features

ZenYT supports all capabilities provided by `yt-dlp`.

## Core Downloading
- Download single videos and audio tracks.
- Download entire playlists or channels.
- Select specific formats and resolutions (e.g., 4K, 1080p, 720p).
- Extract audio only (MP3, M4A, FLAC, WAV).

## Metadata & Extras
- Download subtitles (auto-generated and manual).
- Download thumbnails.
- Embed metadata (chapters, lyrics, video info) into the final file.

## Advanced Usage
- Custom CLI arguments input for power users.
- Network configuration (proxy, user-agents).
- Authentication for private videos.
